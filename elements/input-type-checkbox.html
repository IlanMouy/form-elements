Element : Checkbox

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="checkbox"> permet d'ajouter des cases à cocher.

Un exemple :

Le code derrière tout ça :

<form action="" method="post">
 <label for="checkbox1">:</label>
   <input id="checkbox1" type="checkbox" name="box1"> Voici une box
 <label for="checkbox1">:</label>
   <input id="checkbox2" type="checkbox" name="box2"> Voici une autre
<input type="submit" value="Enregistrer">
</form>

Quelques attributs :

  • label for="" : Une indication pour le navigateur qui renvoie au tag id.
  • id="" : L'indication pour le navigateur et les lecteurs d'écran
